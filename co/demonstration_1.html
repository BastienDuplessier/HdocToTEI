Utilisation du projet

Méthode

Il y a deux manières de lancer le projet :

  1. Avec ant, sans aucun argument (avec le fichier build.xml)

  2. Avec ant, en utilisant le fichier directement (hdoc_to_tei.ant, avec les arguments)

Au préalable

Vérifiez que :

  • Ant est installé

  • Vous avez téléchargé l'archive

  • Vous avez extrait le contenu de l'archive

  • Vous avez une archive Hdoc

Avec build.xml

Marche à suivre

  • Ouvrir une console

  • Aller dans le dossier du programme

  • Modifier les arguments dans le fichier build.xml

  • Lancer ant
